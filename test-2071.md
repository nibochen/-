# 回答如下定义的含义

1.  `int (*ptr)[10];`
    
    回答：声明一个指针ptr指向一个具有10个int元素的一维数组

2.  `(void*)0`
    
    回答：把0强制转换成void类型的指针

3.  `int *f();`
    
    回答：声明一个函数,返回值是指向int的指针

4.  `char *p = (char*)malloc(100);`
    
    回答：分配100个char类型存储单元，并将首地址存储到指针变量p中
